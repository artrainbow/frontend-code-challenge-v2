# React Application music search page

#### Stack:
- Node 6.9.1
- React 16.0.4
- Material-ui 1.2.1
- Jest 23.1.0
- Eslint 4.19.1

#### Data API:
- https://rapidapi.com/deezerdevs/api/deezer-1

#### Installation:
- nvm use 6
- sudo npm install
- nvm use 8
- npm rebuild node-sass

#### Run:
- npm run server
- npm run dev

#### Routes:
- localhost:3000
